# Security Policy

## Data Privacy

- This MCP server fetches RSS feeds directly from public endpoints.
- No authentication or API keys are required.
- No data is collected, stored, or transmitted to any third party.
- Feed configuration is stored locally at `~/.rss-mcp/feeds.json`.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue at https://github.com/lionkiii/rss-feeds-mcp/issues or emailing the maintainer directly.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 72 hours.
